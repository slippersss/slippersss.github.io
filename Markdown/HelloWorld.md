# Markdown

## History

Markdown language, created by John Gruber, 2004.

## Title

Level 1 using "=" or "#".  
Level 2 using "-" or "##".  
Level 3 using "###".  
Level 4 using "####".  
Level 5 using "#####".  
Level 6 using "######".

## Font

*Italic font.*  
**Bold font.**  
***Italic and bold font.***

## Separator

Using "\*\*\*".

## Strikeout

    * Nested list inner item
