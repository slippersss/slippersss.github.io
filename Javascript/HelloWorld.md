# Javascript

## History

Javascript, created by Brendan Eich, 1995, different from java.

## HTML

```html
<script>
    alert("Hello World\n");
</script>
```

## Position

* Javascript in label body will be executed when loading.  
* Javascript in label head will be executed when calling.

## Output

* window.alert()  
* document.write()  
* console.log()  
* \*.innerHTML

## Liberal

* Number  
    * 3.14  
* String

* Expression  
    * 1 + 1  
* Array

    * {id: 0, name: ""}
